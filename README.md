# Project 用の TODO

## 利用目的

友人たちと開発をするため TODO アプリを作ることに

## 開発

- Next.js
- typescript
- vanilla-extract
- firebase
- Material-ui

## 内容

- 個人の TODO
- チーム全体の TODO

個人のものはとりあえずローカルに保存させるようにした。
チーム全体は firebase を用いて保存するようにさせた。
ここからまずは完了未完了のチェックボックス、できれば
アカウントで保存できるようにしたい
